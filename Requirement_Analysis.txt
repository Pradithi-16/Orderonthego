Functional Requirements:
- User Signup/Login
- Browse restaurants/menus
- Search by food/cuisine/location
- Add to cart, order placement
- Payment integration
- Admin and restaurant login
- Order status updates

Non-Functional Requirements:
- Responsive design
- Fast loading speed
- Secure payment gateway
- Scalability to handle traffic

Tech Stack:
Frontend: React.js + Tailwind CSS
Backend: Node.js + Express
Database: MongoDB
APIs: Google Maps, Razorpay
